# SIERRA HEALTH CONNECT

## Overview

Sierra Health Connect is a web-based healthcare information platform designed to improve access to healthcare information and services in Sierra Leone. The website serves as a centralized source of health education, hospital information, emergency contacts, and public health updates.

The project was developed as part of the DCOMP204 Web Design 1 module and aims to address real healthcare challenges faced by citizens by providing easy access to reliable healthcare information.

## Problem Statement

Many citizens in Sierra Leone face difficulties accessing reliable healthcare information and services. People often struggle to:
	•	Find nearby hospitals and clinics.
	•	Access emergency healthcare contacts.
	•	Obtain accurate health information.
	•	Learn about disease prevention.
	•	Stay informed about public health issues.

These challenges can lead to delayed treatment, poor health awareness, and preventable health complications.

## Proposed Solution

Sierra Health Connect provides a digital platform where users can:
	•	Access healthcare information.
	•	Locate healthcare facilities.
	•	View available healthcare services.
	•	Access emergency healthcare contacts.
	•	Learn about disease prevention.
	•	Receive public health updates and alerts.

The platform helps bridge the gap between citizens and healthcare services through technology.

## Project Objectives

The objectives of this project are:
	•	Improve access to healthcare information.
	•	Promote health awareness.
	•	Support disease prevention efforts.
	•	Provide healthcare facility information.
	•	Improve access to emergency healthcare contacts.
	•	Encourage informed healthcare decisions.
